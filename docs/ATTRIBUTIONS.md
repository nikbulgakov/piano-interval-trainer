# Атрибуции

## Salamander Grand Piano V2

- Автор: Alexander Holm.
- Инструмент: Yamaha C5.
- Использование в приложении: локальные MP3-сэмплы для тембра «Пианино».
- Источник: [Tonejs/audio, папка `salamander`](https://github.com/Tonejs/audio/tree/master/salamander).
- Лицензия: [Creative Commons Attribution 3.0 Unported](https://creativecommons.org/licenses/by/3.0/).

Сэмплы используются в сокращённом наборе опорных нот и транспонируются в браузере через Web Audio API.
